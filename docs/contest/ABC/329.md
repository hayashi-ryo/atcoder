# AtCoder Beginner Contest 329

<https://atcoder.jp/contests/abc329>

## A - Spread

- 文字列に対して以下の処理を行う
  - 全ての文字の後に空白を挿入する
  - 最後の空白のみeraseする

## B - Next

- 配列へ挿入・ソート・二番目に大きいものを検索を実施する
  - 逆順ソートをすれば別にfor-loopを逆から動かす必要なかったか

## C - Count xxx

- 求めたいものは、全てのアルファベットごとの最大連続文字数
  - ランレングス圧縮の考え方を利用して配列の作成を行い、数え上げを実施した
